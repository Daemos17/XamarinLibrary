# XamarinLibrary
 Library app using Xamarin Forms and Asp .Net Core
 Основная задача приложения – выдача книги по сканированию QR – кода с помощью телефона.
 
**Авторизация**

Для того чтобы авторизоваться в системе нужно ввести логин и пароль и нажать кнопку Войти:

<p>

**Сканирование QR – кода книги**
<p>
 Для сканирования QR – кода нажмите кнопку Сканировать QR – код, перед вами откроется страница с камерой, если необходимо, предоставьте разрешения для использования приложением камера. Наведите камеру на код, чтобы он находился ровно по центру страницы и нажмите кнопку Сканировать
</p>

 Если процесс сканирования прошел успешно перед вами появится окно выдачи, где вы сможете выбрать количество книг для выдачи, если же QR – код распознать не удалось выведется страница, уведомляющая об этом.
