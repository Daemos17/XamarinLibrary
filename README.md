# XamarinLibrary
 Library app using Xamarin Forms and Asp .Net Core
 Основная задача приложения – выдача книги по сканированию QR – кода с помощью телефона.
 
**Авторизация**

Для того чтобы авторизоваться в системе нужно ввести логин и пароль и нажать кнопку Войти:
